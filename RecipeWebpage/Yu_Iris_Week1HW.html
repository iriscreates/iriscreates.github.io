FAQ

  1. What if I dont have all of these ingredients and/or equipemt?

    2. Some of these ingredients and steps can be swapped or ommited and the drink will still be very tasty. However, it is best to try and use the ingredients listed if you want the same colour and taste as the recipe.

  2. I can't get Empress Gin can I swap it for another gin?

    3. You can change the gin to one you like or have available but the colour will not be present.

  3. I don't have lavender simple syrup what can I subsitute it for?

    4. If you dont have lavender simply syrup you can make it at home with this recipe. If you don't have or want to use lavender syrup I suggest using basic simple syrup.

  4. Can I use regular granulated sugar instead of a syrup?

    5. You can use granulated sugar however, it may not disolve fully into the drink using a simple syrup is best as it does not leave texture in the drink.
